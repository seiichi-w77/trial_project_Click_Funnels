## C2.3: Winning Offerの定義

### 競合の強みから学ぶこと
1. **継続的な質の高いコンテンツ発信**（田中真理子）
2. **ツール・テクノロジーの専門知識**（佐藤健太、高橋誠）
3. **セールスファネル特化の専門性**（山田あゆみ）
4. **分かりやすさと親しみやすさ**（鈴木美咲）
5. **数字・データによる説得力**（山田あゆみ、高橋誠）

### 競合の弱みから差別化する機会
1. **ツール実装の具体的サポート不足** → 実装までの完全サポート（Done With You）
2. **知識提供のみで実践が弱い** → テンプレート・代行サービス提供
3. **高度すぎて難しい or 浅すぎて物足りない** → 中級者向けの実践的内容
4. **女性個人事業主への特化が弱い** → ターゲットを明確化
5. **継続サポートが弱い** → コミュニティとコーチングで継続改善

### 自社のWinning Offer

**「セールスファネル＆DXスターターパック」**

#### 独自価値提案（UVP: Unique Value Proposition）
「知識を学ぶだけでは終わらせない。
あなたの手で実際に動くセールスファネルを構築し、
継続的に改善し続けられる仕組みとコミュニティを提供します」

#### 3つの柱

**1. 実証済みテンプレート＆フレームワーク**
- 業種別ファネル設計図
- 高コンバージョンLP・メールテンプレート
- すぐに使える穴埋め式ワークシート

**2. ツール実装の完全サポート（Done With You）**

## C2.4: ポジショニングの明確化

### 差別化の主要2軸

#### 軸1: サポートレベル（横軸）
- 左端: 情報提供のみ（DIY: Do It Yourself）
- 右端: 完全代行（DFY: Done For You）
- **当社の位置**: 中央やや右（DWY: Done With You - 一緒にやる）

#### 軸2: 専門性の深さ（縦軸）
- 下端: 入門・初心者向け（広く浅く）
- 上端: 上級者向け（深く狭く）
- **当社の位置**: 中央やや上（中級〜上級の実践的内容）
