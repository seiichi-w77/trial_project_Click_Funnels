## C10.1: カスタムドメイン、送信用メールアドレス設定

### カスタムドメイン設定

**推奨ドメイン名**: `funnel-dx.com`

**設定手順**:

1. **ドメイン取得**
   - ドメインレジストラ（お名前.com、GoogleDomains等）で`funnel-dx.com`を取得
   - 年間費用: 約1,500円

2. **ClickFunnelsにドメイン追加**
   - ClickFunnels管理画面 → Settings → Domains
   - 「Add New Domain」をクリック
   - `funnel-dx.com`を入力

3. **DNS設定**
   - ドメインレジストラのDNS設定画面で以下を追加:

   **Aレコード**:
   ```
   ホスト名: @
   値: 199.83.128.123
   TTL: 3600
   ```

   **CNAMEレコード**:
   ```
   ホスト名: www
   値: target.clickfunnels.com
   TTL: 3600
   ```

4. **SSL証明書の有効化**
   - ClickFunnels側で自動的にSSL証明書が発行される（Let's Encrypt）
   - 通常24時間以内に有効化

5. **サブドメイン設定**
   - `lp.funnel-dx.com` → リードマグネットLP用
   - `apply.funnel-dx.com` → 個別相談予約ページ用
   - `members.funnel-dx.com` → メンバーシップサイト用

---

### 送信用メールアドレス設定

**推奨メールアドレス**: `noreply@funnel-dx.com`

**設定手順**:

1. **メールサービスの選択**
   - 推奨: Google Workspace（月額680円/ユーザー）
   - または: SendGrid（無料〜月額$15）

2. **SPFレコード設定**
   - ドメインのDNS設定で以下を追加:
   ```
   タイプ: TXT
   ホスト名: @
   値: v=spf1 include:_spf.google.com include:sendgrid.net ~all
   TTL: 3600
   ```

3. **DKIMレコード設定**
   - Google Workspaceの場合:
   ```
   タイプ: TXT
   ホスト名: google._domainkey
   値: （Google Workspaceから提供される値）
   TTL: 3600
   ```

4. **DMARCレコード設定**
   ```
   タイプ: TXT
   ホスト名: _dmarc
   値: v=DMARC1; p=none; rua=mailto:dmarc@funnel-dx.com
   TTL: 3600
   ```

5. **ClickFunnelsにメールアドレス追加**
   - ClickFunnels管理画面 → Settings → Email Settings
   - 「Add New Sending Email」をクリック
   - `noreply@funnel-dx.com`を入力
   - 認証メールが送信されるので、メール内のリンクをクリック

6. **送信テスト**
   - テストメールを送信し、スパムフォルダに入らないか確認

## C10.3: CRM機能のセットアップ

### CRM初期設定

1. **ClickFunnels CRM有効化**
   - ClickFunnels 2.0の場合、CRM機能は標準搭載
   - Settings → CRM → Enable

2. **パイプライン設定**
   - パイプライン名: `セールスファネル&DX`
   - ステージ設定:

| ステージ | 説明 | 次のアクション |
|---------|------|-------------|
| リード | メアド登録済み | ステップメール育成 |
| 育成中 | ステップメール受信中 | 個別相談誘導 |
| 相談予約済 | 個別相談予約完了 | 事前ヒアリング |
| 相談済 | 個別相談実施済み | 提案書送付 |
| 提案中 | 提案書送付済み | フォローアップ |
| 成約 | 契約締結 | オンボーディング |
| 失注 | 成約せず | 再育成 or リスト削除 |

3. **自動化ルール設定**

**ルール1: リードマグネット登録時**
```
トリガー: LP経由でメアド登録
アクション:
- タグ付与: `lead_magnet_downloaded`
- ステージ: 「リード」に移動
- ステップメール開始
- Slack通知: 「新規リード獲得!」
```

**ルール2: 個別相談予約時**
```
トリガー: 予約ページで日程選択
アクション:
- タグ付与: `consultation_scheduled`
- ステージ: 「相談予約済」に移動
- 事前ヒアリングシート送信
- カレンダー登録
- Slack通知: 「個別相談予約!」
```

**ルール3: 成約時**
```
トリガー: 申し込みページで決済完了
アクション:
- タグ付与: `customer_standard`（プラン別）
- ステージ: 「成約」に移動
- ウェルカムメール送信
- 契約書送信
- Slack通知: 「新規顧客獲得!」
- チーム全員にお祝いメッセージ
```

## C10.4: HTML形式で各ランディングページ（LP）の構成案作成

### LP構成の基本方針

- **デザインコンセプト**: スティーブ・ジョブズとジョナサン・アイブの視点で、先進的でスタイリッシュ
- **カラー**: ネイビー（#1a2b4a）× オレンジ（#ff6b35）× ホワイト（#ffffff）
- **フォント**: ヒラギノ角ゴ / Noto Sans JP（和文）、San Francisco / Inter（欧文）
- **ミニマリズム**: 余白を多く、シンプルで分かりやすい
- **モバイルファースト**: スマホで見やすいレスポンシブデザイン

1. Stripeアカウント作成
2. ClickFunnels → Settings → Payments → Add Stripe
3. Stripe APIキーを入力
4. テスト決済実施

**対応カード**:
- VISA
- Mastercard
- JCB
- American Express

#### 2. PayPal連携（PayPal決済）

**設定手順**:
1. PayPalビジネスアカウント作成
2. ClickFunnels → Settings → Payments → Add PayPal
3. PayPal APIキーを入力

### 決済ページの構成

1. **ヘッドライン**: 「あなた専用のセールスファネルを、30日で一緒に作りましょう」
2. **プラン選択**: ラジオボタンでスタンダード/プレミアム/VIP選択
3. **支払い方法選択**: 一括 or 分割
4. **顧客情報入力**:
   - 氏名
   - メールアドレス
   - 電話番号
   - 住所（請求書送付先）
5. **決済情報入力**:
   - クレジットカード番号
   - 有効期限
   - CVV
6. **利用規約**: チェックボックス
7. **CTA**: 「今すぐ申し込む」ボタン
