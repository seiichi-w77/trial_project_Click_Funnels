# C5: 商品の体系化

## C5.1: オファーの体系化

C4で整理したオファーを基に、一貫性のある商品ラインナップとして体系化します。

## 商品体系の一貫性

### 一貫性の3つの軸

#### 軸1: 段階的な価値提供

すべての商品は、「学ぶ → 作る → 売る」という一貫した価値提供プロセスに基づいています。

**無料オファー**: 学ぶ（診断・教育）
**個別相談**: 設計（あなた専用の計画）
**有料プラン**: 作る（Done With You実装）
**継続プログラム**: 売る（運用・改善・拡大）

#### 軸2: Done With Youの一貫性

すべての有料プランで、「一緒に作る」スタイルを貫いています。

- 教材だけ渡して終わり ❌
- 完全代行で丸投げ ❌
- **一緒に作って、スキルも習得** ✅

#### 軸3: 成果へのコミットメント

すべてのプランで、具体的な成果目標を設定し、保証を提供しています。

- リード獲得数の明確な目標
- 90日間リード獲得保証
- 満足度保証

## 商品体系の差別化ポイント

### 差別化ポイント1: 明確な価格体系

**一般的な競合**:
- 「お問い合わせください」
- 価格が不透明
- 追加費用が発生しやすい

**私たち**:
- 全プラン価格を明示
- 追加費用なし（広告費除く）
- 透明性の高い価格設定

---

### 差別化ポイント2: 成果保証

**一般的な競合**:
- 「教えるまで」が仕事
- 成果は保証しない
- 返金なし

**私たち**:
- 「成果が出るまで」が仕事
- 90日間リード獲得保証
- 30日間満足度保証（全額返金）

---

### 差別化ポイント3: 継続的な関係

**一般的な競合**:
- プログラム終了 = 関係終了
- 継続サポートは高額

**私たち**:
- VIP継続プログラム（月10万円）
- コミュニティで継続的な交流
- アップデート情報の共有

## LTV（顧客生涯価値）最大化の仕組み

### 典型的な顧客のLTV計算

**パターン1: スタンダード → 継続 → 追加ファネル**
```
スタンダードプラン: 55万円
 ↓ (90日後)
VIP継続プログラム: 11万円/月 × 12ヶ月 = 132万円
 ↓ (1年後)
追加ファネル構築: 50万円

合計LTV: 237万円
```

**パターン2: プレミアム → 継続**
```
プレミアムプラン: 110万円
 ↓ (180日後)
VIP継続プログラム: 11万円/月 × 24ヶ月 = 264万円

合計LTV: 374万円
```

**パターン3: VIP → 継続 → 追加ファネル × 2**
```
VIPプラン: 198万円
 ↓ (365日後)
VIP継続プログラム: 10万円/月 × 24ヶ月 = 240万円
 ↓
追加ファネル構築: 80万円 × 2回 = 160万円

合計LTV: 598万円
```

**平均LTV目標**: 150万円以上（C0のKGI達成）

## 商品体系の運用方針

### 方針1: 価値のはしご（Value Ladder）を登ってもらう

**目標**:
- すべての顧客を「次のステップ」に案内
- スタンダード → 継続 → 追加ファネル → プレミアム
- 常に「次の成長」を提案

### 方針2: 無理な売り込みをしない

**基本姿勢**:
- 顧客が成果を出してから、次のステップを提案
- 「売上のため」ではなく、「顧客の成功のため」
- 断られたら、無理に勧めない

### 方針3: データに基づいた改善

**定期レビュー**:
- 各プランの成約率を月次でチェック
- LTV・CAC比率を四半期でレビュー
- 商品内容を継続的にアップデート

### 方針4: 顧客の声を反映

**フィードバック収集**:
- プログラム終了時にNPSアンケート
- 改善要望を四半期ごとに集計
- 年1回、商品内容を大幅アップデート
