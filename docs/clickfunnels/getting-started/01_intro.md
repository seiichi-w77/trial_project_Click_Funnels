# ClickFunnels API Documentation Overview

## Welcome & Introduction

The ClickFunnels API documentation welcomes developers with the statement: "We are excited to present you with the ClickFunnels API, which currently entails the V2 REST API, webhooks, and integrations."

### Core Capabilities

The platform enables developers to:
- Import and export data between ClickFunnels and external applications
- Extend ClickFunnels functionality through custom applications
- Leverage OAuth 2.0 to perform actions on behalf of other users

## Documentation Structure

The docs are organized into two main sections:

**Guides Section** covers:
- Getting Started materials
- REST API fundamentals (authentication, rate limiting, requests/responses)
- Data handling (filtering, pagination, time zones)
- Business operations (orders, fulfillments, products)
- Funnel-specific features
- Webhooks and integrations
- No-code/low-code solutions

**API Reference** provides detailed endpoint documentation accessible separately.
