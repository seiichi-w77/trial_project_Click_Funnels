# ClickFunnels 2.0 Integration Guide

## Overview

ClickFunnels 2.0 offers an "API and webhook ecosystem to create functionality on top of the ClickFunnels platform, either for yourself or for our mutual customers."

## Integration Methods

### 1. **Internal Integration (Self-Use)**
For personal integrations, developers can directly leverage API keys and webhook functionality without additional approval. Navigate to the [Getting Started](/docs/getting-started) section to begin implementation.

### 2. **Third-Party Developer Integration**
External developers seeking to serve mutual customers should:
- Submit a developer account application through [ClickFunnels Support](https://support.myclickfunnels.com/)
- Include detailed use case descriptions
- Reference the integration documentation
- Expect engineering team review and response

Alternative: Post in the discussion forum with contact information, though applications through support receive priority handling.

### 3. **Public Integration Program**
Successful integrations serving customers may qualify for:
- Addition to the official integrators list
- Community spotlight across ClickFunnels channels (Facebook, Reddit, native community)
- Featured placement in API announcements

## Notable Integrators

The platform lists active integrators including: OfferLab, Zendrop, Morrison Brothers Store Builder, SegMetrics, Wake Up Warrior, WickedReports, Funnels on Rails, Everflow, and Make.

## Core Documentation Areas

- **Authentication & Security**: OAuth 2.0, SSO, API authentication protocols
- **API Fundamentals**: Rate limiting, request/response formats, filtering, pagination
- **Webhooks**: Event configuration, signature verification, event type handling
- **Business Logic**: Orders, fulfillments, products, funnels
- **No-Code Solutions**: Available for non-developers

## Next Steps

Review authentication documentation and contact support for developer account applications.
