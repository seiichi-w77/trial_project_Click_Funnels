# コアマーケティングメッセージの定義

## 1. エグゼクティブサマリー

本ドキュメントは、「GitHub as OS」という技術ビジョンと、「AI Dreams Factory」という事業ビジョンに基づき、コアマーケティングメッセージを定義するものです。AI開発のあり方を根底から変革する自律型AIエージェントフレームワーク「Miyabi」をエンジンとし、日中米のクリエイターエコシステムを繋ぐ「AI Dreams Factory」を事業の器とすることで、単なるツール提供に留まらない、次世代のAI産業創出ムーブメントを牽引します。

本メッセージングは、技術的優位性と生産性の飛躍的向上を求めるトップティアの開発者、および自らのアイデアを迅速に事業化したいテクノロジー起業家を主要ターゲットとし、彼らの野心と課題に直接的に響く言葉で構成されています。

---

## 2. コアビジョン：2つの柱

我々のメッセージは、以下の2つの相互補完的なビジョンを基盤とします。

###  Pillar A: GitHub as OS (技術ビジョン)
- **コンセプト:** GitHubを単なるコードリポジトリではなく、OS（オペレーティングシステム）として捉え直す。GitHub Issueを「命令」として入力すれば、自律型AIエージェント群「Miyabi」が開発の全工程（設計、実装、テスト、レビュー、デプロイ）を実行し、最終成果物（Pull Request）を「出力」する、という新しい開発パラダイム。
- **価値:** 開発者を反復的なタスクから解放し、人間はより高次元の「ビジョン定義」や「創造的判断」に集中できる。開発サイクルの劇的な短縮と生産性の指数関数的向上を実現する。

### Pillar B: AI Dreams Factory (事業ビジョン)
- **コンセプト:** アイデア創出からプロダクト開発、事業化、収益化、イグジットまでを一気通貫で支援する「AIプロダクトの生産ライン」。個々のクリエイターや開発者が持つ優れたアイデアや技術を、ビジネスとして成立させるためのマーケティング、営業、コミュニティ連携といったリソースを提供するエコシステム。
- **価値:** 技術力はあっても事業化のノウハウが不足している才能を支援し、「AIで生計を立てる」クリエイターを量産する。特に、日中米の巨大AIコミュニティ（WAY to AGI, Shift AI等）と連携し、グローバルな共創と市場開拓を加速させる。

## 3. ターゲットオーディエンス

**主要ターゲット：野心的なテクノロジー・ビルダー**

- **属性:** AIエンジニア、ソフトウェア開発者、データサイエンティスト、技術系スタートアップ創業者、企業のR&D担当者。
- **特徴:**
    - 技術の最先端を常に追求し、アーリーアダプターである。
    - 生産性、効率性、インパクトを至上の価値とする。
    - 既存の開発手法に限界を感じ、パラダイムシフトを求めている。
    - ハッカソンのような競争と共創の場で輝くタイプ。
    - 自らの手で革新的なプロダクトを創り出し、世界に影響を与えたいという強い意欲を持つ。

## 5. トーン・オブ・ボイス

*   **Visionary (ビジョナリー):** 未来の働き方、未来の産業のあり方を提示する。
*   **Authoritative (権威的):** 裏付けされた技術と実績に基づき、自信を持って語る。
*   **Technical (技術的):** 専門用語を恐れず、ターゲットに響く的確な言葉を選ぶ。
*   **Empowering (力を与える):** 開発者やクリエイターの可能性を最大限に引き出す存在として語る。
*   **Collaborative (協調的):** コミュニティと共に創り上げるオープンな姿勢を示す。
